| ------------------------- | ---------------------------------------- | ---- |

| [cmd]  /?                 | 查看该命令的帮助                                 |      |
| @echo  on/off             | 是否在输入命令时显示出完整输入路径                        |      |
| [cmd]  /A                 | 与[cmd]  /a是一个意思,windows不区分大小写            |      |
| 帮助中[/A \| /M]\[/D[:date]] | []表示可选,\|表示或,所以/D[:date]也就可以为/D或/D:m-d-y的形式了 |      |
| rem [comment]             |                                          |      |
| [cmd] > [output file]     |                                          |      |
| [cmd] < [input file]      |                                          |      |
| [cmd] \| more             |                                          |      |
| %1,%2                     |                                          |      |
| xcopy                     |                                          |      |
| dir                       |                                          |      |
| *                         | wildcard,通配符                             |      |
| cls                       | clear screen,清屏                          |      |
| esc                       | 清除所有当前输入行的内容                             |      |
